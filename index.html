Spedmo Vario Testing Interface

Congratulations, you have successfully setup your Vario Testing Interface.

You can now build your own customised IOS/Android interface for your flights and competitions.

To get started use your browser developer toolbar to change the presentation mode to ether a mobile or tablet device.

Some useful links for development which are included as functionality you can call in this testing harness...

  • Bootstrap Framework
  • Font Awesome
  • Using Jquery
  • Pressure JS
  • Google Maps API

GPS & NMEA specific

  • GPS.js
  • NMEA sentence information
  • NMEA Generator

Should you wish to use additional Javascript libraries, call and initalise them from your init.js by including them in the /src/js folder rather than editing the Testing Interface.

NMEA file Reader

Select a NMEA formatted file to input into simulate device input, you can create your own or you can find some example content in the /sample-data folder.
